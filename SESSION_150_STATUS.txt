WHAT NOODLE 2 IS
════════════════

Noodle 2 is a comprehensive **safety-aware, policy-driven orchestration
system** for large-scale physical-design experimentation built on OpenROAD.

Key Capabilities:
- Safety domains (sandbox, guarded, locked) with ECO class constraints
- Multi-stage refinement with deterministic survivor selection
- Case lineage tracking with DAG-based provenance
- Ray-based distributed execution across clusters
- Docker-based trial isolation with OpenROAD
- Comprehensive telemetry and artifact management
- PDK support (Nangate45, ASAP7, Sky130)
- Professional reporting and visualizations
- 20+ structured error codes (E1001-E1030) with actionable suggestions
- Publication-quality heatmap generation (300+ DPI)
- JSON-LD metadata for research publication

Production Quality:
- 3,139 automated tests (100% passing)
- Full type hints with mypy validation
- Comprehensive error handling and recovery
- Detailed logging and tracing
- Git integration for reproducibility
- Backward-compatible telemetry schemas

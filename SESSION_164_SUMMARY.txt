WHAT IS NOODLE 2?
─────────────────

Noodle 2 is a safety-aware, policy-driven orchestration system for large-scale
physical design experimentation with OpenROAD. It provides:

CORE FEATURES:
• Safety domains with ECO class constraints
• Multi-stage refinement with survivor selection
• Deterministic case lineage tracking (DAG-based)
• Policy-driven early stopping and resource management
• Ray-based distributed execution
• Docker-based tool isolation

QUALITY ATTRIBUTES:
• 3,139 automated tests (100% passing)
• Full type hints with mypy validation
• Comprehensive error handling and recovery
• Detailed logging, tracing, and telemetry
• Git integration for reproducibility
• JSON-LD metadata for research publication

SUPPORTED PDKs:
• Nangate45 (open-source educational)
• ASAP7 (academic 7nm)
• Sky130 (SkyWater 130nm)

## What Is Noodle 2?

**Noodle 2 v0.1.0** is a comprehensive safety-aware orchestration system for
large-scale physical design experimentation with OpenROAD.

### Key Features

**Safety-Aware Execution:**
- Three safety domains: sandbox, guarded, locked
- ECO class constraints and validation
- Run legality reporting
- Policy-driven abort criteria

**Multi-Stage Refinement:**
- Sequential stage execution
- Survivor selection between stages
- Stage-specific execution modes
- Trial budgets and resource management

**Distributed Execution:**
- Ray-based task distribution
- Docker container isolation
- Resource tracking and management
- Graceful shutdown and checkpointing

**Comprehensive Telemetry:**
- Artifact indexing and organization
- JSON-LD metadata for research publication
- Git integration for reproducibility
- Heatmap generation and visualization

**PDK Support:**
- Nangate45 (complete)
- ASAP7 (complete, with failure detection)
- Sky130A (complete)

### Production Quality

- **3,139 automated tests** (100% passing)
- **Full type hints** with mypy validation
- **Comprehensive error handling** and recovery
- **Detailed logging** and tracing
- **Self-documenting artifacts** with README generation
- **Docker-based isolation** for reproducibility

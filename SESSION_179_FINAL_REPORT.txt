## What Noodle 2 Provides

Noodle 2 is a comprehensive **safety-aware orchestration system** for large-scale physical design experimentation with OpenROAD, featuring:

### Core Capabilities
1. **Safety Domains** - Three-tier safety model (sandbox, guarded, locked)
2. **Multi-Stage Refinement** - Progressive design optimization with survivor selection
3. **Case Management** - Deterministic lineage tracking with DAG-based versioning
4. **Policy Engine** - Configurable early stopping and resource management
5. **Distributed Execution** - Ray-based parallel processing
6. **Artifact Management** - Comprehensive telemetry and result tracking

### Technical Features
- Docker-based OpenROAD integration (efabless/openlane:ci2504-dev-amd64)
- PDK support: Nangate45, ASAP7, Sky130
- ECO framework with safety classification
- Git integration for reproducibility
- JSON-LD metadata for research publication
- Comprehensive error handling and recovery

### Quality Assurance
- 3,139 automated tests (100% passing)

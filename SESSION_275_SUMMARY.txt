WHAT NOODLE 2 IS
================================================================================

Noodle 2 is a comprehensive safety-aware orchestration system for large-scale
physical design experimentation with OpenROAD. It provides:

**Core Architecture:**
- Safety-critical experiment controller
- Policy-driven decision engine
- Deterministic control plane for PD workflows
- Structured experimentation framework

**Key Features:**
- Safety domains (sandbox, guarded, locked)
- Multi-stage refinement pipelines
- Survivor selection and lineage tracking
- Trial budgets and early stopping
- Ray-based distributed execution
- Docker containerized tool execution
- Comprehensive artifact management
- JSON-LD metadata for reproducibility

**Supported Technologies:**
- OpenROAD (placement, routing, timing)
- OpenSTA (static timing analysis)
- Docker containers (isolated execution)
- Ray (distributed orchestration)
- PDKs: Nangate45, ASAP7, Sky130

**Design Philosophy:**
- Does not replace PD tools
- Provides safety and control layer
- Enables structured experimentation
- Maintains auditability
- Enforces determinism
- Prevents silent escalation of risk

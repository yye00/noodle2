## FEATURE: TELEMETRY JSON FORMATTING

### What It Does
Ensures all telemetry JSON files are professionally formatted with:
- Pretty-printed JSON (2-space indentation)
- ISO 8601 timestamps (YYYY-MM-DDTHH:MM:SSZ)
- Human-readable durations (e.g., "1h 2m 5s")
- Descriptive field names (snake_case)
- Proper newlines at end of files
